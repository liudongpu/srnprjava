中国驻阿富汗大使馆昨日表示，在阿富汗首都喀布尔8日发生的中国公民遇害失踪事件中，两失踪人员均为男性，目前其中一人已成功脱险。使馆向阿警方了解，警方正全力搜索另一名失踪者，尚未有明确消息。目前确认的中国公民遇难人数仍为3人。据新华社 死者 遇害中国人为个体户 据新华社电 中国驻阿富汗大使馆昨日说，8日阿首都喀布尔中国公民遇害失踪事件中的一名失踪者，使馆已将其安置在安全地点。 使馆表示，此案中的3名遇害者和两名失踪者均为在阿个体经营户，遇害者为一名男性和两名女性，其中两名女性是8日当天前往现场勘查情况时遇到凶徒而不幸遇害。在案发现场，还发现一具阿富汗公民遗体。 在中国驻阿使馆不断敦促和交涉下，阿富汗警方正全力缉拿凶手。喀布尔警局犯罪调查科主任查希尔向记者表示，警方已组成专案组，争取早日抓住凶手。 使馆表示，事件如有进展，将及时发布信息。使馆同时提醒在阿中方人员提高警惕，注意外出安全。 中国驻阿富汗大使馆9日对记者说，8日发生的中国人在阿首都喀布尔遇害失踪事件目前正在进一步调查中，事件性质尚不能确定。尚不能确定是否特意针对中国人。目前进一步细节尚不便透露。
